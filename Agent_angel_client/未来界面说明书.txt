未来界面说明书

1. 登录方式演进：
   - 当前：网页版登录 (Web Client)。
   - 未来：PC 客户端 (.exe) 或 安卓/iOS 原生应用。

2. 分布式算力架构：
   - 网页版用户：
     - 算力来源：服务器 (Server-Side Compute)。
     - 限制：受限于服务器 GPU/CPU 资源，高并发下可能排队。
     - 适用：临时使用、手机浏览器访问。
   
   - PC 客户端用户 (.exe)：
     - 算力来源：本地电脑 (Local Compute)。
     - 机制：客户端内置 Playwright 和 Python 环境，直接在本地运行隐形浏览器。
     - 优势：
       1. 无限并发：每个用户消耗自己的算力。
       2. 零延迟：无需视频流传输，直接本地渲染。
       3. 隐私更好：浏览数据不经过服务器渲染。
     - 适用：重度用户、挂机任务、高性能需求。

3. 混合云调度：
   - 服务器将自动识别登录来源。
   - 如果是 .exe 登录，服务器仅下发指令 (JSON)，不进行渲染。
   - 如果是 Web 登录，服务器负责渲染并推送视频流。
