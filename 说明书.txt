⚡️ 操作流程总结
安装环境：进入 Angel_Server，运行 setup.bat（仅需一次）。
启动后端：进入 Angel_Server，运行 start_server.bat（挂起 8000 端口）。
启动前端：进入 Angel_Client，运行 start_client.bat（挂起 5500 端口并弹出浏览器）。

第五步：服务器防火墙设置 (关键！)
在 Windows Server 上，默认防火墙会拦截 8000 端口。如果不开放，你的客户端（.html）将无法连接。
按 Win + R，输入 wf.msc 打开高级防火墙设置。
点击左侧 “入站规则” (Inbound Rules) -> 右侧 “新建规则” (New Rule)。
选择 “端口” (Port) -> 下一步。
选择 TCP，特定本地端口输入：8000 -> 下一步。
选择 “允许连接” (Allow the connection) -> 下一步 -> 下一步。
名称输入：Angel Server 8000 -> 完成。
