## 1. 🆕 新增应用：性能调优 (Performance Tuner)

### 🎯 目的
为了解耦复杂的设置逻辑，我创建了一个独立的应用程序模块 `performance.js`，专门用于管理系统性能和兼容性设置。

### 🛠️ 功能亮点
1.  **🚀 性能模式切换：**
    *   **高性能 (60FPS)：** 画面丝滑流畅，开启抗锯齿。
    *   **节能模式 (30FPS)：** 降低刷新率和分辨率，适合笔记本省电。
    *   *实时生效，无需重启。*

2.  **🐢 强制无 GPU 兼容模式：**
    *   专为显卡驱动有问题或 WebGL 崩溃的用户设计。
    *   开启后，强制小天使使用 CPU 渲染（软件渲染）。
    *   *注意：切换此选项会自动提示刷新页面以生效。*

3.  **🚑 紧急修复 (重置小天使)：**
    *   直接集成了之前开发的重置指令。
    *   一键将小天使恢复出厂设置（位置归零、状态重置）。

## 2. 🔗 深度集成与解耦

### 🛠️ 代码变更
*   **`js/apps/performance.js`**: 全新的应用文件，包含完整的 UI 和逻辑。
